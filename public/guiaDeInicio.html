Introducción

LayBlu UI es una libreria de componentes de libre uso para crear aplicaciones web simples y hermosas

Inicio rápido

Para usar LayBlu UI en tus proyectos puedes utilizar jsDelivr, un CDN gratuito de código abierto. Tambien puedes optar por incluir la descarga en tu proyecto a traves de nuestros servidores

CSS

Copie y pegue el siguiente CDN de la hoja de estilo link en su < head > antes que todas las demás hojas de estilo.

<link href="https://cdn.jsdelivr.net/npm/layblu-ui@1.0/dist/css/layblu.min.css" rel="stylesheet">

JS

Copie y pegue el siguiente <script> justo antes de la etiqueta de cierre </body>, para poder habilitar el funcionamiento de los componentes que nuestra libreria requiere

<script src="https://cdn.jsdelivr.net/npm/layblu-ui@1.0/dist/js/layblu.min.js"></script>
